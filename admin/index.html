How to publish changes

  1. Edit your content here.
  2. Click Export JSON.
  3. In GitHub, replace content/portfolio-content.json with the exported file.
  4. For images, upload them into assets/uploads/ and use that path in the editor, for example assets/uploads/project1.png.
  5. For your CV, upload the PDF into assets/cv/ and set the CV URL.
GitHub Pages is static hosting. This editor cannot directly write to GitHub without GitHub OAuth/server setup. This export-based workflow is the fastest safe method. A direct Save-to-GitHub button can be added later.
